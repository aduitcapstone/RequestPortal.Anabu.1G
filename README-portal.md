# SmartBrgy — Resident Portal
**Barangay Anabu I-G, Imus City, Cavite**

Online portal para sa mga residente — mag-request ng barangay documents.

## Live Portal

🌐 [Buksan ang Portal](https://[iyong-username].github.io/smartbrgy-portal/portal.html)

## Mga Serbisyo

- 📄 Barangay Clearance
- 🏠 Certificate of Residency
- 📋 Certificate of Indigency
- 🪪 Barangay ID
- 💼 First Time Jobseeker Certificate
- 🏪 Business Clearance

## Para sa mga Residente

1. Pumili ng uri ng dokumento
2. Punan ang form
3. I-submit — makukuha ang reference code
4. Pumunta sa Barangay Hall na may reference code at valid ID

## Technical

Ang portal ay kumokonekta sa backend API sa `https://smartbrgy-api.onrender.com`.
